Featured Project // Project Lead & Lead Dev

HEXILEXI — AI WORD DETECTIVE

An LLM-driven active vocabulary mentor component designed to replace passive dictionary definitions with interactive learning modes for primary school readers. Integrated as a custom React component within the university's public STREEN platform.

  • • Architected Prompt Pipeline: Designed a two-step "Architect and Builder" system to process inputs using Gemma 3:12b via local Ollama API instances, minimizing response latencies and maintaining child-friendly persona consistency.
  • • Algorithmic Evaluation: Programmed a local answer-grader using Levenshtein distance fuzzy-matching, ensuring precise validation for kids' spelling deviations without resource-heavy LLM calls.
  • • UX Testing & Stability: Managed UI state-locking and rate-limiting request queues to handle concurrent clicks during real-world primary school tests.
